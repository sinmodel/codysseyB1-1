## 2. 날짜별 요약

### 7/6(월)

### 7/7(화)

- 보고서 제출 지연 문제로, 앞으로는 **매월 마지막 영업일 오전 초안 작성 / 오후 검토 후 제출** 프로세스로 정리.

### 7/8(수)
- 행사 일정은 **8/27로 연기될 가능성이 높음**, 금요일까지 최종 회신 받기로 함.
- **행사 안내문 초안 작성 후**, 디자인팀이 **배너 시안과 신청 페이지 이미지 작업** 진행 예정.
- 신청 페이지는 오픈 전 **최소 1일 이상 테스트**하기로 함.
- 예산이 빠듯해 **사은품 축소 또는 외부 촬영 생략** 검토 중이며, 추후 재논의 예정.
- 행사 소개 문구에는 **“실무자 대상” 표현을 포함**하되, 딱딱하지 않게 다듬기로 함.
